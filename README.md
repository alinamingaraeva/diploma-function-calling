# Function Calling Demo

## Описание инструмента
**search_knowledge_base** – поиск по базе знаний техподдержки. Параметры: `query` (обязательный), `product` (опционально).

## Результаты тестирования

### Тест A (требует вызова функции)
**Запрос:** «Подскажи, как сбросить пароль?»  
**Поведение модели:** Вызвала функцию `search_knowledge_base` с аргументами `{"query": "сбросить пароль", "product": "AcmeCloud"}`.  
**Результат функции:** «Извините, не удалось найти информацию...» (из-за несовпадения формулировок).  
**Финальный ответ:** Модель вежливо сообщила, что не нашла инструкцию, и предложила переформулировать вопрос.  

### Тест B (не требует вызова)
**Запрос:** «Привет! Как дела?»  
**Поведение модели:** Не вызвала функцию, ответила текстом.  
**Финальный ответ:** «Здравствуйте! У меня все хорошо, спасибо. Как я могу Вам помочь?»  

### Тест C (пограничный случай)
**Запрос:** «Расскажи что-нибудь о своём сервисе»  
**Поведение модели:** Модель решила, что это общий вопрос, не требующий поиска, и не вызвала функцию.  
**Финальный ответ:** Развёрнутое описание сервиса AcmeCloud.
