**search_knowledge_base** – поиск по базе знаний техподдержки. Параметры: `query` (обязательный), `product` (опционально).

**Поведение модели:** Вызвала функцию `search_knowledge_base` с аргументами `{"query": "сбросить пароль", "product": "AcmeCloud"}`.
